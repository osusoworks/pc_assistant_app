# SENPAI シンプルUIデザイン仕様書

## 📋 デザイン分析（添付画像ベース）

### 🎨 視覚的特徴
- **ミニマルデザイン**: 余計な装飾を排除したシンプルな構成
- **モノクロベース**: 白背景に黒文字、グレーの境界線
- **フラットデザイン**: 影やグラデーションを使わない平面的なデザイン
- **明確な境界**: 細い線による要素の区切り
- **適度な余白**: 要素間の十分なスペーシング

## 🏹 矢印オーバーレイ改善仕様

### 🎯 現在の問題点
1. **影の存在**: 不要な影効果が表示されている
2. **先端の三角形**: 短くてつぶれた形状
3. **視認性**: シンプルUIに合わない装飾的なデザイン

### ✨ 改善仕様

#### **矢印デザイン**
```python
# 基本仕様
線幅: 3px
色: #FF4444 (鮮やかな赤)
影: なし
アンチエイリアス: 有効

# 矢印の形状
全長: 60px
軸線: 40px
三角形部分: 20px (軸線の50%)
三角形幅: 16px (軸線の40%)
```

#### **描画仕様**
```python
# 矢印の構成要素
1. 軸線 (Line): 開始点から三角形基部まで
2. 三角形 (Polygon): 先端の矢印部分

# 座標計算
軸線開始: (x, y)
軸線終了: (x + 40*cos(angle), y + 40*sin(angle))
三角形頂点: (x + 60*cos(angle), y + 60*sin(angle))
三角形左基部: 軸線終了点から垂直方向に8px
三角形右基部: 軸線終了点から垂直方向に-8px
```

#### **描画パラメータ**
```python
# Canvas設定
fill: #FF4444 (三角形の塗りつぶし)
outline: #FF4444 (線の色)
width: 3 (線幅)
smooth: True (アンチエイリアス)

# 影効果の除去
shadow: None
drop_shadow: False
blur: 0
```

## 🔄 実装方針

### 📝 ファイル構成
```

overlay_module_clean.py       # 改善された矢印オーバーレイ
main_controller_simple.py     # シンプル版メインコントローラー
run_simple_clean.py          # 新しい実行ファイル
```

### 🎨 UIフレームワーク
- **ベース**: tkinter (標準ライブラリ)
- **スタイル**: ttk.Style() でカスタマイズ
- **レイアウト**: grid() レイアウトマネージャー使用
- **レスポンシブ**: ウィンドウリサイズ対応

### 🏹 矢印描画
- **キャンバス**: tkinter.Canvas
- **描画メソッド**: create_line() + create_polygon()
- **座標計算**: 数学的な角度計算
- **アニメーション**: なし（静的表示）
